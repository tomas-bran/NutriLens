# El catálogo

Cada producto que analizás queda guardado. El **Catálogo** te deja encontrarlo rápido.

![Catálogo con filtros y tarjetas de producto](/img/screens/catalogo.png)

## Recorrerlo

- Cada producto es una **tarjeta** con su imagen, nombre, categoría, **badge de riesgo** (bajo/medio/alto) y chips de alérgenos.
- Tocá una tarjeta para abrir su **[ficha completa](/funcionalidades/resultado)**.
- El botón **Nuevo análisis** te lleva directo a **[Analizar](/funcionalidades/analizar)**.

## Filtrar y buscar

Arriba tenés todo para acotar la lista:

- **Buscar** por nombre.
- **Categoría** (galletitas, snacks, bebidas, lácteos, cereales…).
- **Riesgo** (bajo / medio / alto).
- **Alérgeno** (excluir o encontrar por alérgeno).
- **Aptitud** (vegano / celíaco / sin lactosa).

Los filtros se **combinan** (se aplican todos juntos) y quedan en la URL, así podés compartir o volver a la misma vista.

## "Analizados por vos"

Arriba hay un selector **Todos / Analizados por vos**. El catálogo es una base compartida; con **"Analizados por vos"** ves **solo los productos que vos analizaste**.

![Catálogo filtrado por "Analizados por vos"](/img/screens/catalogo-mios.png)

:::tip
Este conteo es el que ves en tu perfil. Si un producto tuyo se elimina, **deja de contar** y desaparece de tu lista — ver **[Administración](/administracion)**.
:::
