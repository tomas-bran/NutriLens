# NutriWorld (beta)

Tu góndola, en 3D. Un supermercado virtual donde **NutriLens, en forma de robot**, interpreta lo que pedís en lenguaje natural, te **guía a la góndola correcta** y resalta los productos aptos.

![NutriWorld — supermercado 3D con el robot guía](/img/screens/nutriworld.png)

:::warning Acceso restringido (beta)
NutriWorld es una **experiencia beta solo para administradores**. Si no sos admin, el ítem **no aparece** en el menú y la ruta no está disponible. Ver **[Administración](/administracion)**.
:::

## Cómo se usa

1. Entrá a **NutriWorld** desde el menú (visible solo para admins).
2. Esperá la pantalla de carga; cuando termina, el robot NutriLens **aparece con una animación de entrada**.
3. **Movés a tu avatar** con `W A S D` o las flechas (`Shift` para correr). La cámara lo sigue.
4. Escribí qué buscás en la barra de abajo (p. ej. _"Mostrame productos veganos"_) o tocá una **sugerencia**.
5. NutriLens **camina hasta la góndola** correcta y **resalta** los productos que cumplen.
6. Acercate a un producto y tocá **`E`** (o hacé click) para ver su **ficha**.

## Qué tiene

- **Avatar + NPC con IA:** el robot usa la **misma IA del chat** para interpretar tu pedido.
- **Pills dinámicas:** las sugerencias cambian según lo que vas hablando.
- **Minimapa:** abajo a la derecha, con tu posición y la góndola objetivo resaltada.
- **Cuatro góndolas:** Sin TACC, Vegano, Sin lactosa y Snacks.
- Podés **ver cualquier producto** (click/`E`), hayas preguntado o no.

:::note
NutriWorld es una demo de hacia dónde puede ir el producto. Los productos del mundo 3D son un set curado de ejemplo.
:::
