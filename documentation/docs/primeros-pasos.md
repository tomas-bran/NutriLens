# Primeros pasos

## Requisitos

- Un navegador moderno (Chrome, Edge, Firefox o Safari), desde compu o celular.
- Una **cuenta de Google** para ingresar.
- Una foto o PDF de la etiqueta del producto que quieras analizar.

## 1. Ingresá

Abrí la app en **[nutrilens-app.azurewebsites.net](https://nutrilens-app.azurewebsites.net)** y entrá con tu cuenta de Google.

![Pantalla de ingreso](/img/screens/login.png)

El ingreso es necesario para que tu **catálogo** y tus **conversaciones** queden guardados y sean solo tuyos.

## 2. Conocé el recorrido

Una vez adentro, la barra lateral (o la barra inferior en el celular) te lleva a todo:

| Sección        | Para qué sirve                                                |
| -------------- | ------------------------------------------------------------- |
| **Inicio**     | Bienvenida y accesos rápidos.                                 |
| **Analizar**   | Subir la foto de una etiqueta y analizarla.                   |
| **Catálogo**   | Ver todo lo que analizaste, con filtros.                      |
| **Chat**       | Preguntarle al asistente sobre tus productos.                 |
| **NutriWorld** | Supermercado 3D _(beta — visible solo para administradores)_. |

## 3. Analizá tu primer producto

El flujo principal es simple:

1. Andá a **Analizar**.
2. Subí la foto del producto (y, si querés, el código de barras).
3. Tocá **Analizar** y mirá el resultado.

Seguí con la guía paso a paso de **[Analizar un producto](/funcionalidades/analizar)**.

:::tip
¿Querés probar sin analizar nada todavía? Entrá a **Catálogo** para ver productos de ejemplo y al **Chat** para hacerle preguntas.
:::
